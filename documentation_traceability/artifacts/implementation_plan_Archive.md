# Project Implementation Plan & Strategy

## 1. Architecture
- 3-Tier decoupled services (Nginx, Angular, .NET).
- Clean Architecture (Core, Infrastructure, API).

## 2. Deployment
- Docker Compose orchestration.
- AWS EC2 target.
- ZIP-based high-speed file transfer.

## 3. Security
- SSH Keys secured via icacls.
- .env injection for secrets.
- GitIgnore scrubbing.

## 4. Verification
- Playwright E2E tests.
- Browser recordings.
- Health checks.
